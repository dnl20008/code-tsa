LỘ TRÌNH LÀM BÀI

5 bước đơn giản để sở hữu mã code thi thử

1
Mua mã Code

Kéo xuống dưới cùng trang này, nhập Email và thanh toán 25.000đ để nhận mã kích hoạt.

2
Truy cập Website

Vào trang chủ kỳ thi TSA của ĐH Bách Khoa: tsa.hust.edu.vn

3
Đăng ký thi

Chọn mục "Thi thử TSA" trên menu, sau đó bấm Đăng ký (Tạo tài khoản nếu chưa có).

4
Nhập mã Code

Hệ thống yêu cầu mã kích hoạt. Nhập mã Code bạn vừa mua ở Bước 1 vào để mở khóa bài thi.

5
Làm bài & Xem điểm

Làm bài trong thời gian quy định. Nộp bài để xem ngay điểm số và đáp án chi tiết.
